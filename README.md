# CV Interactif – Abdoul DIOP

## Description
Page web personnelle présentant mon curriculum vitae de manière interactive.
Réalisée dans le cadre du module **Développement Web – L2 Informatique**,
Université Iba Der THIAM de Thiès.

## Technologies utilisées
- **HTML5** – Structure sémantique (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- **CSS3** – Mise en page (Flexbox & Grid), responsive design, animations, thème sombre/clair
- **JavaScript (Vanilla)** – Interactivité sans aucune bibliothèque externe

## Fonctionnalités JavaScript
1. **Mode sombre / clair** – Bascule avec sauvegarde dans `localStorage`
2. **Menu hamburger** – Navigation mobile avec ouverture/fermeture
3. **Animation des barres de compétences** – Déclenchée au scroll via `IntersectionObserver`
4. **Formulaire de contact** – Validation côté client (champs vides, format e-mail)
5. **Bouton retour en haut** – Apparaît après 300px de scroll

## Comment ouvrir le projet
1. Télécharger ou cloner le dossier `mon-cv/`
2. Ouvrir `index.html` dans un navigateur (Chrome ou Firefox recommandés)
